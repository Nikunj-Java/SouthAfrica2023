Pipes Example

There are Two Types of pipes available in angular

  1. Inbuilt-Pipe
  2. Custom Pipe

1. Inbuilt Pipe

A. String Pipe

UPPERCASE:{{name | uppercase}}

lowercase:{{name | lowercase}}

B. Number Pipe

pi:{{pi | number:'5.5-5'}}

pi:{{pi | number:'.3-3'}}

pi:{{pi | number:'.10-10'}}

C. Currency Pipe

USD:{{a | currency : 'USD':true: '3.3-3'}}

USD:{{a | currency : 'USD':false: '.3-3'}}

EURO:{{b | currency : 'EURO':true: '.3-3'}}

EURO:{{b | currency : 'EURO':false}}

INR:{{b | currency : 'INR':true: '.3-3'}}

INR:{{b | currency : 'INR':false}}

D. Date Pipe

Date:{{today}}

Date:{{today | date}}

Date:{{today | date: 'dd/MM/yyyy'}}

E. Object Pipe

Without JSON pipe:{{object}}

Without JSON pipe:{{object | json}}

2. Custom Pipe

Original String:{{cust}}

Custom pipe:{{cust | changestring:' '}}
